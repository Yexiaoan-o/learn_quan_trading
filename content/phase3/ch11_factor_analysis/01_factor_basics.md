## 量化因子的基本概念

量化因子（Factor）是现代量化投资的核心语言——它是一种能够解释和预测股票横截面收益差异的量化指标。简单来说，因子回答了"为什么有的股票涨、有的股票跌"的问题。

---

### 一、什么是量化因子

在因子投资（Factor Investing）的框架中，每只股票可以被分解为一系列因子暴露（Factor Exposure）的组合：

```
股票预期收益 = 因子1 × 因子暴露1 + 因子2 × 因子暴露2 + ... + 残差收益
```

每个因子代表一种系统性的风险溢价来源，而因子的"暴露"则衡量该股票在这一风险维度上的高低。因子投资的核心假设是：**具有相似因子特征的股票，未来收益表现也会相似**。

因子研究的基本流程：

1. **因子构建** → 提出经济逻辑，设计计算公式
2. **因子测试** → 检验因子对未来收益的预测能力
3. **因子组合** → 将多个有效因子合成为一个综合信号
4. **策略实施** → 基于因子信号构建投资组合

### 二、常见因子分类

#### 2.1 价值因子（Value Factors）

价值因子衡量股票"便宜还是贵"。核心思想是买入被市场低估的股票、卖出被高估的股票。

| 因子名称 | 计算公式 | 含义 |
|----------|----------|------|
| **市盈率（PE）** | 股价 / 每股收益 | 越低越"便宜"，典型价值指标 |
| **市净率（PB）** | 股价 / 每股净资产 | <1 时理论上股价低于清算价值 |
| **股息率（Dividend Yield）** | 每股分红 / 股价 | 越高代表股东回报越丰厚 |
| **企业价值/EBITDA** | EV / EBITDA | 考虑了债务水平的企业估值 |
| **自由现金流收益率** | FCF / 市值 | 衡量公司产生现金的能力 |

```sql
// 市盈率因子（PE）—— 最低PE的股票最有价值
select
    symbol,
    close / eps_ttm as pe_ratio
from stock_fundamentals
where date = latest
order by pe_ratio asc   // PE越低越有投资价值
```

#### 2.2 动量因子（Momentum Factors）

动量效应是金融市场中最稳健的异象之一：过去表现好的股票倾向于继续表现好。

| 因子名称 | 计算方式 | 说明 |
|----------|----------|------|
| **价格动量** | 过去(N-M)个月的累计收益（跳过最近M个月） | 学术界标准：过去12个月（跳过最近1个月） |
| **残差动量** | 剔除市场和行业影响后的超额收益动量 | 比原始动量更纯净 |
| **盈利动量** | 分析师盈利预测的上调/下调幅度 | 基本面动量 |

```sql
// 12-1个月价格动量因子
select
    symbol,
    (close / mavg(close, 252)[21] - 1) * 100 as momentum_12m1     // 领先最近252个交易日中从第21天前的价格
from daily_data
context by symbol
order by date
```

> **注意**：价格动量短期（1个月内）往往表现为反转效应（Reversal），因此学术界标准做法是跳过最近1个月，计算"12-1"动量。

#### 2.3 质量因子（Quality Factors）

质量因子衡量公司的经营质量和财务稳健性。高质量公司通常具有更稳定的盈利和更低的破产风险。

| 因子名称 | 计算公式 | 含义 |
|----------|----------|------|
| **净资产收益率（ROE）** | 净利润 / 净资产 | 股东权益回报率，越高越好 |
| **资产负债率** | 总负债 / 总资产 | 越低财务风险越小 |
| **毛利率** | (收入-成本) / 收入 | 衡量公司定价权 |
| **应计项目** | (净利润-经营现金流) / 总资产 | 越低盈利质量越好 |
| **资产周转率** | 收入 / 总资产 | 经营效率指标 |

```sql
// 质量因子：高ROE + 低负债率
select
    symbol,
    net_income / total_equity as roe,
    total_liabilities / total_assets as debt_ratio,
    case
        when net_income / total_equity > 0.15 and
             total_liabilities / total_assets < 0.5 then '高质量'
        when net_income / total_equity < 0 then '低质量'
        else '中等'
    end as quality_label
from stock_fundamentals
```

#### 2.4 波动率因子（Volatility Factors）

低波动率异象是金融学中的著名发现：低波动率的股票长期收益反而高于高波动率股票。

| 因子名称 | 计算公式 | 说明 |
|----------|----------|------|
| **市场Beta** | Cov(R_stock, R_market) / Var(R_market) | 系统性风险暴露 |
| **特质波动率** | 残差标准差（剔除市场影响后） | 公司特有风险 |
| **下行波动率** | 负收益的标准差 | 只衡量下跌风险 |
| **最大回撤** | 历史最大峰值到谷值的跌幅 | 尾部风险指标 |

```sql
// 波动率因子：计算60日历史波动率和Beta
select
    symbol,
    mstd(log(close / prev(close)), 60) * sqrt(252) as volatility_60d,    // 年化波动率
    mbeta(close, market_close, 252) as beta                              // 滚动Beta
from daily_data
where date = latest
```

### 三、因子的构建流程

一个高质量的因子从提出到投入使用需要经过系统的构建过程：

| 步骤 | 内容 | 注意事项 |
|------|------|----------|
| **1. 经济逻辑** | 为什么这个因子能预测收益？是风险补偿还是行为偏差？ | 必须有合理的因果解释 |
| **2. 数据获取** | 确定所需数据源（价量、财务、另类数据） | 注意数据质量和覆盖度 |
| **3. 因子计算** | 编写公式，处理缺失值和异常值 | 中位数替代或Winsorize截尾 |
| **4. 行业中性化** | 剔除行业差异的影响 | 使用行业虚拟变量回归取残差 |
| **5. 市值中性化** | 剔除市值大小的影响 | 防止大市值股票的因子值系统性偏高 |
| **6. 标准化** | 将因子值转为Z-score（均值0、标准差1） | 使不同因子之间可比 |

```sql
// 因子标准化：转为Z-score
def factorStandardize(factorValues){
    return (factorValues - avg(factorValues)) / std(factorValues)
}

// 因子中性化：剔除行业和市值影响后取残差
// 使用 ols 函数完成回归取残差
def factorNeutralize(factorValues, industry, marketCap){
    // 对行业做one-hot编码后进行回归
    residual = ols(factorValues, [ones(size(factorValues)), industry, log(marketCap)])
    return residual
}
```

### 四、因子投资的核心理念

| 理念 | 说明 |
|------|------|
| **收益来源** | 因子收益是承担系统性风险的补偿（风险溢价） |
| **分散化** | 多因子组合比单因子更稳健，不同因子在不同市场环境中表现各异 |
| **长期性** | 因子策略需要长期坚持，短期可能失效（因子拥挤、市场环境变化） |
| **动态性** | 因子表现随市场环境变化，需要持续监控和迭代 |

> **关键认知**：量化因子既是"显微镜"（帮助理解收益来源），也是"工具"（构建投资策略）。有效的因子必须同时具备经济意义上的合理性和统计意义上的显著性。仅靠数据挖掘（Data Snooping）发现的模式，在样本外大概率失效。
