## 3.1 K线图的构成：OHLC

K线图（Candlestick Chart）是金融市场中最基本、最核心的可视化工具。每一根K线记录了一个固定时间段内的四个关键价格：

| 价格 | 英文 | 简称 | 含义 |
|------|------|------|------|
| 开盘价 | Open | O | 该时间段内第一笔成交价 |
| 最高价 | High | H | 该时间段内最高成交价 |
| 最低价 | Low | L | 该时间段内最低成交价 |
| 收盘价 | Close | C | 该时间段内最后一笔成交价 |

这四个价格合称为 **OHLC**，是所有金融时间序列分析的数据基础。在量化交易中，OHLC 数据通常以 CSV 文件或数据库表的形式存储，每一行代表一根K线的时间区间（如1分钟、1小时、1天）。

```
          最高价 ┐
              ┌─┼─High
              │ │
    上影线 → │ │
              │ │
  实体 → ┌───┐ │
         │   │ │
         │   │ ├─ 实体(阳线/阴线)
         └───┘ │
              │ │
    下影线 → │ │
              │ │
          最低价 └─┼─Low
```

K线的四个要素在量化策略中各有用途：**开盘价**反映隔夜信息和集合竞价博弈；**最高价**和**最低价**共同决定当日价格运行的区间宽度；**收盘价**是最重要的价格——多数技术指标基于收盘价设计，且机构结算通常参考收盘价。

## 3.2 阳线与阴线

K线分为两大类：

| 类型 | 判别条件 | 通常颜色 | 市场含义 |
|------|----------|----------|----------|
| 阳线（Bullish） | 收盘价 > 开盘价 | 红色/空心（A股）或绿色（美股） | 买方力量占优，价格上涨 |
| 阴线（Bearish） | 收盘价 < 开盘价 | 绿色/实心（A股）或红色（美股） | 卖方力量占优，价格下跌 |

> **注意颜色差异**：A股市场习惯用红色表示上涨（阳线），绿色表示下跌（阴线）；而美股恰好相反。分析数据时务必确认颜色规范，避免误判。

### 常见K线形态的量化意义

**(1) 长阳线 / 长阴线**
实体长度远大于上下影线，表示单边趋势强劲。在量化策略中，长阳线常被用作突破信号的确认条件。

**(2) 十字星（Doji）**
开盘价约等于收盘价，上下影线较长。反映多空力量均衡，市场方向不明。量化策略中常被作为趋势反转的辅助信号。

**(3) 锤头线 / 吊颈线**
实体较小（在顶部或底部），一侧有长影线。出现在下跌趋势末端的锤头线可能预示着反转向上，但单独使用误判率较高，需要结合成交量和其他指标确认。

```
十字星：    锤头线：    长阳线：
  ┼         ┐          ┌─High
  │         │          │
  │         └┐         │
  │          └┐        │
  │           └┐       │
  ┼       ──────┘      └─Low
```

> **量化视角**：不要试图"主观解读"K线形态。正确的做法是将形态特征编码为量化条件（如：实体/影线比例 > 3 → 强趋势信号），在历史数据中验证其统计显著性。

## 3.3 成交量与换手率

### 成交量（Volume）

成交量是同一时间段内成交的股票数量（手数或股数），是市场活跃度的直接度量。在K线图中通常以柱形图显示在价格下方。

**量价关系法则**：
- **价涨量增**：趋势健康，买方积极入场
- **价涨量缩**：上涨乏力，可能即将回调
- **价跌量增**：抛压沉重，下跌加速
- **价跌量缩**：卖盘枯竭，底部信号

### 换手率（Turnover Rate）

换手率是衡量股票流动性和市场参与热情的重要指标：

```
换手率 = 当日成交量 / 流通总股本 × 100%
```

| 换手率范围 | 市场状态 | 策略参考 |
|-----------|---------|---------|
| < 1% | 冷门股，交易清淡 | 不适合短线策略 |
| 1% - 3% | 正常交易区间 | 中低频策略可接受 |
| 3% - 7% | 交易活跃 | 适合多数量化策略 |
| 7% - 15% | 高度活跃 | 短线/Momentum策略理想 |
| > 15% | 异常活跃 | 可能有重大事件，注意风险 |

> 换手率过低的股票滑点大、冲击成本高，量化策略应尽量避免。一般建议选择日均换手率在 1% 以上的标的。

## 3.4 估值指标：市盈率与市净率

### 市盈率（PE Ratio）

市盈率是股价与每股盈利的比值，反映市场对公司未来盈利的估值溢价：

```
市盈率（静态）= 当前股价 / 上一年度每股收益（EPS）
市盈率（TTM）= 当前股价 / 最近四个季度每股收益之和
市盈率（动态）= 当前股价 / 预测未来年度每股收益
```

| PE水平 | 传统解读 | 量化应用 |
|--------|---------|---------|
| < 10 | 估值偏低 | 价值因子选股 |
| 10 - 20 | 估值合理 | 中性判断 |
| 20 - 30 | 估值偏高 | 成长因子参考 |
| > 30 | 高估 | 需结合行业判断 |

量化策略中，PE可被纳入多因子模型，但要注意行业差异——科技股的PE普遍高于银行股，跨行业直接比较无意义。

### 市净率（PB Ratio）

市净率是股价与每股净资产的比值：

```
市净率 = 当前股价 / 每股净资产
```

PB < 1 意味着股价低于公司净资产，俗称"破净"。PB 对资产密集型行业（银行、地产、钢铁）的估值判断效果优于 PE，但对轻资产公司（软件、服务）的参考价值较低。

## 3.5 核心指标汇总表

| 指标 | 公式/含义 | 用途 | 注意点 |
|------|----------|------|--------|
| OHLC | 开/高/低/收盘价 | K线基础数据 | 不同频率代表不同策略 |
| 阳线 | Close > Open | 多头信号 | A股/美股颜色相反 |
| 阴线 | Close < Open | 空头信号 | 需结合成交量判断 |
| 成交量(Volume) | 成交股数/手数 | 活跃度度量 | 注意复权对量的影响 |
| 换手率 | Vol/流通股本 | 流动性指标 | < 1% 不适合短线 |
| 市盈率(PE) | 股价/EPS | 估值水平 | 行业差异大，须行业内比较 |
| 市净率(PB) | 股价/每股净资产 | 资产估值 | 适合重资产行业 |

## 3.6 数据存储格式

在实际量化系统中，OHLCV 数据通常以如下格式存储：

```python
# 标准股票日线数据示例
"""
日期,      开盘,   最高,   最低,   收盘,    成交量,    成交额
2024-01-02, 15.20, 15.85, 15.10, 15.67,  12500000, 195000000
2024-01-03, 15.70, 15.95, 15.50, 15.72,   9800000, 154000000
2024-01-04, 15.72, 15.75, 15.20, 15.30,  11200000, 173000000
"""
```

在 DolphinDB 等专业数据库中，还会增加复权因子（复权因子）、交易状态（停牌标志）、涨跌幅等辅助字段。高质量的数据管理是量化交易的基础——垃圾数据进，垃圾结果出（Garbage In, Garbage Out）。
